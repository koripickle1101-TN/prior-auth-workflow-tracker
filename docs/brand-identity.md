# Brand identity

## Primary colors

White: #FFFFFF

Tennessee Orange: #FF8200

Black: #000000

Warm gray: #E8E1D8

Soft gray: #F7F4EF

Text gray: #4B4B4B

## Visual direction

The prior authorization workflow tracker uses a premium healthcare operations identity.

The design should feel:

- professional
- editorial
- feminine
- structured
- operationally intelligent
- Tennessee-rooted
- clean enough for recruiter and hiring manager review

## Layout rules

Use a white background.

Use black for primary text.

Use Tennessee Orange for accents, active states, KPI emphasis, node borders, and priority markers.

Use warm gray for borders, separation, and soft structure.

Use 40 to 50 percent whitespace when possible.

Use structured grids with intentional asymmetry.

## Typography direction

Use high-contrast editorial serif headlines.

Use clean sans-serif supporting text.

Recommended headline stack:

Georgia, Times New Roman, serif

Recommended body stack:

Inter, Arial, sans-serif

## Visual elements

Use double-ring circular nodes with soft orange glow.

Use dotted or fading connector lines between workflow stages.

Use white cards with warm gray borders.

Use orange top borders on KPI cards.

Keep tables clean and readable.
